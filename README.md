📌 Overview

The Lap7 Examination System is a console-based application developed using C# and Object-Oriented Programming (OOP) concepts.
The system simulates a simple examination environment where exams, questions, and students are managed through different classes and structures.

This project demonstrates how OOP principles such as Inheritance, Abstraction, Polymorphism, and Encapsulation can be used to design and implement an examination system.

The system allows creating exams, starting them, answering questions, and displaying exam results.

🎯 Features

Create and manage exams

Support for different exam types

Manage questions and answers

Allow students to take exams

Show exam results

Use of events and delegates for exam lifecycle management

Demonstration of advanced OOP design


📚 Learning Objectives

This project helps in practicing:

OOP design

Class relationships

Events and delegates

Inheritance and polymorphism

Building structured C# applications
